Created by: E. Tarleton edmund.tarleton@materials.ox.ac.uk

3D FEM code using linear 8 node element with 8 integration pts (2x2x2) per element.

   4.-------.3
   | \       |\
   |  \      | \    my
   1.--\---- .2 \
    \   \     \  \
     \  8.--------.7
      \  |      \ |  mz
       \ |       \|
        5.--------.6
             mx
y   ^z
 ↖  |
  \ |
   \|---->x
Note

This is rotated about the x axis w.r.t. to the local (s1, s2, s3) system. calc_σHat uses custom linear shape functions that eliminate the need for a Jacobian, speeding up the calculation. We keep the Jacobian in this function because it's only run at simulation initialisation so it's not performance critical, which lets us use standard shape functions.
